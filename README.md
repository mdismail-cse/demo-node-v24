# Node 24 Deployment Test App

A simple, modern Node.js application built to test deployment pipelines.

## Features
- **Zero Dependencies**: Uses native `node:http` module.
- **ES Modules**: Modern JavaScript syntax.
- **Premium UI**: Sleek, glassmorphic landing page included in the response.
- **Environment Info**: Displays Node version, platform, and uptime.

## Getting Started

### Prerequisites
- Node.js (Version 24 recommended, works on 18+)

### Running Locally
```bash
npm start
```
The server will be available at `http://localhost:3000`.

## Deployment
This app is designed to be easily deployable. It reads the `PORT` environment variable provided by most hosting services.
